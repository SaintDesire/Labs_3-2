age: Возраст человека.
sex: Пол человека.
cp: Тип боли в груди (0 - типичная стенокардия, 1 - атипичная стенокардия, 2 - неангинозная боль, 3 - бессимптомное)
trtbps: Покойное кровяное давление (в мм рт.ст.).
chol: Уровень холестерина в мг/дл.
fbs: Уровень сахара в крови натощак (> 120 мг/дл) (1 - да; 0 - нет).
restecg: Результаты электрокардиографии в покое (0 - нормальный, 1 - наличие аномалии волн ST-T, 2 - гипертрофия левого желудочка по критериям Эстеса).
thalachh: Максимальная достигнутая частота сердечных сокращений.
exng: Стенокардия, вызванная физической нагрузкой (1 - да; 0 - нет).
oldpeak: Уровень депрессии ST, вызванный физической нагрузкой по сравнению с отдыхом.
